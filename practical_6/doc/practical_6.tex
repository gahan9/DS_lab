\begin{itemize}
	\item Iterator is used to iterate over all data from file storage.
	\item For projection program won't use any condition to filter out data and displays all the data which exists in data file.
	\item For select operation conditions are to be matched as case insensitive (default implementation) and supports case sensitive operation if optional parameter passed to function.
	\item Program also contains function of creating dummy data as much as needed.
	\item Easier to use for small scale system without any overhead of setup and configuration.
	\item Program is made modular so that it's functionality can be further updated vary easily for large scale.
